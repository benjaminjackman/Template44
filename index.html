Template44 - A CoffeeScript Templating Library

Template44 - A CoffeeScript DSL for HTML Templates

Template44 is a CoffeeScript DSL for HTML templates that supports server side and client side rendering.

Template44 is open-source, hosted on GitHub, and uses the MIT License (just like coffeescript).

My First Template

Template44 elements can quickly be given ids with #some-id, or classes with .some-css-class.

#A Very Simple Hello World Template
myFirstTemplate = Template44 ($$) ->
  $$ 'div#Hello.red', {}, ->
    $$ '', "Hello World!"

When this is run:

myFirstTemplate()

Will Produce: 


  Hello World!
